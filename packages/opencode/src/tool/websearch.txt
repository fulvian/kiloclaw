- Search the web via the Knowledge provider catalog.
- Provider priority is Tavily, Firecrawl, Brave, DDG, then Wikipedia fallback.
- Returns fresh information for current events and post-cutoff topics.
- Supports configurable result counts and provider selection (`auto` by default).
- Always returns provider metadata, fallback chain, and per-provider errors for auditability.

Usage notes:
  - Prefer `provider: "auto"` to use the best available provider.
  - You may force a provider with `provider: "tavily" | "firecrawl" | "brave" | "ddg" | "wikipedia"`.
  - If API keys are missing for paid providers, the tool falls back to free providers.
  - The tool output includes links and snippets suitable for synthesis and fact-checking.

The current year is {{year}}. You MUST use this year when searching for recent information or current events
- Example: If the current year is 2026 and the user asks for "latest AI news", search for "AI news 2026", NOT "AI news 2025"
